这款插件把之前我写的一些插件和 @airyland 大大写的 [SF-Notifier][1] 插件融合在一块而创造出来的。名字来源于“豆瓣精灵”这个插件。目前插件的主要功能有如下几个：

#### 0.0.1版
- 【站内消息浏览器提醒扩展】
    修复了之前 @airyland 版本点击消息无法去除提醒的问题。
- 【用户已回答答案搜索】
    后台选项中可单独开启/关闭该功能，是 [SF Answer Search][2] 的融合版。
- 【在线聊天室】
    后台选项中可单独开启/关闭该功能，是 [SF ChatOnline][3] 的融合版。
- 【回答区图片粘贴上传】
    后台选项中科单独开启/关闭该功能。
